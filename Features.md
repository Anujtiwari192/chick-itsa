# Features

## Admin
1. Login
   - Go to `/login` and use the admin credentials.
2. Create doctors
   - Admin Dashboard → Create Doctor form → Submit.
3. Create patients
   - Admin Dashboard → Create Patient form → Submit.
4. Assign patients to doctors
   - Admin Dashboard → Assign Patient to Doctor → Submit.
5. Edit doctors, patients, and assignments
   - Admin lists → click `Edit` → Save.
6. Delete doctors, patients, and assignments
   - Admin lists → click `Delete`.
7. Search and filter
   - Use Search + Show filters on Admin Dashboard.
8. CSV export
   - Export Doctors / Patients / Assignments buttons.

## Doctor
1. Login
   - Go to `/login` using doctor credentials.
2. View profile
   - Doctor Dashboard → Profile section.
3. Search assigned patients
   - Search box on Doctor Dashboard.
4. Add patient
   - Doctor Dashboard → Add Patient → Submit (auto-assigns to doctor).
5. Update assigned patient details
   - Assigned Patients list → Update → Save.
6. Add patient history entry
   - Update Patient page → History Entry → Save.
7. View patient history log
   - Update Patient page → History Log table.

## Patient
1. Login
   - Go to `/login` using patient credentials.
2. View own profile (read-only)
   - Patient Dashboard → Profile section.
3. View assigned doctor(s)
   - Patient Dashboard → Assigned Doctor(s) table.
